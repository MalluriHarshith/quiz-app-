# Quiz Application #

## Description
This is a full-stack quiz application built using React (frontend) and Spring Boot (backend).  
It supports Admin and Student roles.

-----------------------
## Features
- Admin can add, edit, delete questions
- Students can take quizzes
- Results are stored in database
- Prevent multiple attempts for same test
- Admin can delete results to allow reattempt

------------------------
## Tech Stack
- Frontend: React.js
- Backend: Spring Boot
- Database: MySQL

-----------------------
# How to Run

## Backend
1. Open backend project
2. Run Spring Boot application
3. Runs on: http://localhost:8080
4. or Directly run QuizAppApplication.java

## Frontend
1. Open quiz-frontend folder
2. Run:
   npm install
   npm start
3. Runs on: http://localhost:3000
